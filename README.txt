
# Smart Expense Tracker - Final Version

## Prerequisites:
- Node.js installed
- Internet connection (for MongoDB Atlas)

## Setup Instructions:

### 1. Frontend (React)
```
cd client
npm install
npm start
```

### 2. Backend (Node.js + Express + MongoDB Atlas)
- Add your MongoDB Atlas URI in `server/.env` as:
```
MONGO_URI=your_mongodb_atlas_connection_string
```
Then run:
```
cd server
npm install
npm start
```

### App will be running on:
Frontend: http://localhost:3000
Backend: http://localhost:5000

Happy Tracking!
